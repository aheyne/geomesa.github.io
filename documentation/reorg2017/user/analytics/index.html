Analytics with GeoMesaÂ¶

GeoMesa provides several resources for analytics on spatio-temporal data.

  • GeoMesa Spark allows execution of jobs on Apache Spark using data stored in GeoMesa Accumulo or any other GeoTools-backed data store.
  • GeoMesa Processes describes a set of analytic WPS processes for use in GeoServer for heatmap generation, KNN searches, statistical sampling, and more.
  • GeoMesa Stream Processing provides tools to process streams of SimpleFeatures.
