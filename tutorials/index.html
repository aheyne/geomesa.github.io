Tutorials

The tutorials below describe how to set up GeoMesa; ingest, query, and export data in a variety of contexts; perform a variety of data analyses; and accomplish a variety of other tasks. Most of the tutorials include example Java code that demonstrate the different capabilities of the system.

Getting Started

  • GeoMesa Deployment - How to set up GeoMesa, including the command line tools and the GeoServer plugin.
  • Quickstart - How to create a collection of of new geospatial records in Accumulo and query and visualize the results.
  • Kafka Quickstart - How to configure GeoMesa to produce and consume messages in Apache Kafka and visualize the results.
  • GeoMesa GeoServer UI - More details about the GeoMesa User Interface in GeoServer.

Data In / Out

  • Ingest & Export with GeoMesa Tools - How to ingest and export geospatial data using the GeoMesa command line tools.
  • Streaming Ingest - How to ingest big geospatial data into GeoMesa via an Apache Kafka/ Storm stream.
  • Map-Reduce Ingest - How to ingest big geospatial data via a Hadoop map-reduce job.
  • Raster Data - How to work with raster data in GeoMesa.
  • Management with GeoMesa Tools - How to use the GeoMesa command line tools for data feature management.
  • GeoMesa Transformations - How to use GeoMesa to perform relational transformations (projections) on geospatial data.

Data Analysis

  • Apache Spark Analysis - How to use GeoMesa with the Apache Spark data processing engine.
  • WPS Tube Select - How to use GeoMesa with GeoServer's Web Processing Services (WPS) to run time-interpolated queries.

Security

  • GeoMesa Authorizations - How to set visibilities on GeoMesa data in Accumulo and apply authorizations to queries.
  • GeoMesa Feature Level Visibility - How to set visibilities on features in GeoMesa Accumulo data.

Indexing and Queries

  • Geohash Ranges - How the GeoMesa query planner identifies the Geohash prefixes that cover a query polygon.
